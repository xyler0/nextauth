# X Poster Frontend

Simple OAuth testing dashboard.

## Setup

```bash
cd frontend
npm install
```

## Run

```bash
npm start
```

Visit: http://localhost:3001

## Features

- ✅ OAuth login with GitHub/Twitter
- ✅ Account linking
- ✅ Repository selection
- ✅ Connection status
- ✅ Logout

## Usage

1. Click "Sign in with GitHub" or "Sign in with Twitter"
2. Authorize the app
3. You'll be redirected back with a JWT token
4. View and manage connected accounts
5. Select repositories to monitor
